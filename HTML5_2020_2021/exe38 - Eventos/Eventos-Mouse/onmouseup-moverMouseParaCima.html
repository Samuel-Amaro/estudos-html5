Eventos Mouse

Atributo de evento HTML onmouseup

Exemplo: Execute um JavaScript ao liberar um botão do mouse sobre um parágrafo:

Clique no texto!
Mantenha o botão do mouse pressionado por alguns segundos, depois solte!
A função mouseDown() é acionada quando o botão do mouse é pressionado sobre este parágrafo. A função define a cor do texto para vermelho. A função mouseUp() é disparada quando o botão do mouse é liberado. A função mouseUp() define a cor do texto como verde.
