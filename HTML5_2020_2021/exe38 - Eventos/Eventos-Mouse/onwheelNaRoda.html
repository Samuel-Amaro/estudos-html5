Atributos de evento HTML onwheell

Exemplo: Execute um JavaScript quando o usuário rolar a roda do mouse sobre um elemento <div>:

Este exemplo demonstra como atribuir um evento de evento "onwheel" a um elemento DIV. Role a roda do mouse sobre mim - para cima ou para baixo!

Uma função é acionada quando você rola a roda do mouse sobre div. A função define o tamanho da fonte de div para 35 pixels.
