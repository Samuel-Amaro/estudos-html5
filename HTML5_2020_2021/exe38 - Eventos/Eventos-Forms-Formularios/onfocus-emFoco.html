Evento onfocus(em foco)

ExemploExecute um JavaScript quando um campo de entrada receber o foco:

Uma função é acionada quando um dos campos de entrada obtém o foco. A função altera a cor de fundo do campo de entrada.


Quando você insere o campo de entrada, uma função é acionada que define a cor de fundo para amarelo. Quando você sai do campo de entrada, uma função é acionada que define a cor de fundo para vermelho.
