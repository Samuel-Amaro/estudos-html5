Atributos dos Inputs - Atributos dos elementos Input

Cada elemento input possui atributo, que permite mais manipulação sobre o elemento input, e tem mais poder de personalizaçaõ sobre o elemento

Atributo de valor - attribute value

Campos de entrada - input type:text - com valore iniciais(padrão), o proprio programador seta valores na caixa de texto inicialmente, para mostrar o uso do atributo value


Atributo de somente leitura - attribute readonly

um input type text, so vai poder fazer leitura isso e, so mostra saida de dados, não recebe entrada de dados do usuario, e uma caixa de texto, um campo de texto


Atributo desativado - attribute disabled

o campo de texto, desativado, não pode ser clicado nem receber entrada e nem saida de dados, input type: text


O atributo de tamanho - attribute size

permite controlar o tamanho de um input, especifica a largura visivel em caracteres de um input de tipo text,tel,url,email,password


O Atributo Comprimento Maximo - attribute maxlength

especifica em um input type text o numero maximo de caracteres permitidos em um campo de entrada


Os Atributos mínimo e máximo - attibutes min and max

os atributos minimo e maximo, juntos conseguem especificar um intervalo de numeros, nos seguintes inputs - number, range, date, datetime-local, month, time, week



O Atributo Múltiplo - attribute multiple

o atributo multiple especifica inserir mais de um valor em um campo de entrada com email e file, segure o CTRL + clique nos arquivos para selecionar mais de um arquivo

O atributo padrão - attribute pattern

Trabalha com expressões regulares

O Atributo placeholder

O atributo necessário - attribute required

o attribute required, diz que o input tem que ter um entrada de dado do usuario, ele e de preechimente de valor obrigatorio

O atributo de etapa - attribute step

O Atributo Foco automático - attribute autofocus

no campo do primeiro nome, ao carregar a pagina, o cursor vai estar focado ja nesse campo, automaticamente
