O atributo de prenchimento automatico(autocomplete)

O atributo autocomplete especifica se um campo de entrada deve ou não ter o autocomplete habilitado.

Preencha e envie o formulário e recarregue a página para ver como o preenchimento automático funciona.

Observe que o preenchimento automático está "ativado" para o formulário, mas "desativado" para o campo de e-mail!
