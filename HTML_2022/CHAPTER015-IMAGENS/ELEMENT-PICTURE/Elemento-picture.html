O elemento picture

Redimensione o navegador para ver diferentes versões da imagem sendo carregada em diferentes tamanhos de janela de visualização. O navegador procura o primeiro elemento de origem em que a consulta de mídia corresponde à largura da janela de visualização atual do usuário e busca a imagem especificada no atributo srcset.

O elemento img é necessário como a última tag filho do bloco de declaração de imagem. O elemento img é usado para fornecer compatibilidade com versões anteriores para navegadores que não suportam o elemento picture ou se nenhuma das tags de origem corresponder.

Observação: o elemento de imagem não é compatível com IE12 e versões anteriores ou Safari 9.0 e versões anteriores.
