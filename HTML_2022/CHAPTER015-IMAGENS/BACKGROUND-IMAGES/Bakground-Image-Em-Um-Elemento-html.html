Imagen de Fundo

Uma imagen de fundo para o elemento p:

Você pode especificar imagens de fundo
Para qualquer elemento HTML visível.
Neste exemplo, a imagen de Fundo
é especificado para um elemento p.
Po padrão, a imagen de fundo
se repetirá na(s) direções
onde é menor que o elemento
onde é especificado. (Tente redimensionar o
janela do navegador para ver como o
imagende fundo se comporta.)
