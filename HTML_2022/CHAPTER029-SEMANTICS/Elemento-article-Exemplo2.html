Google Chrome

O Google Chrome é um navegador da Web desenvolvido pelo Google, lançado em 2008. Atualmente, o Chrome é o navegador da Web mais popular do mundo!

Mozzila Firefox

Mozilla Firefox é um navegador web de código aberto desenvolvido pela Mozilla. O Firefox é o segundo navegador mais popular desde janeiro de 2018.

Microsoft Edge

O Microsoft Edge é um navegador da Web desenvolvido pela Microsoft, lançado em 2015. O Microsoft Edge substituiu o Internet Explorer.
